# Longest Palindromic Substring

**Difficulty:** Medium

## Problem Statement

Given a string `s`, return the longest palindromic substring in `s`.

## Examples

### Example 1

```text
Input: s = "babad"
Output: "bab"
Explanation: "aba" is also a valid answer.
```

### Example 2

```text
Input: s = "cbbd"
Output: "bb"
```

## Constraints

- `1 <= s.length <= 1000`
- `s` consists of only digits and English letters.
